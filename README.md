## Installing zsh

zsh is packaged for all major operating systems, including most Linux
distributions, macOS, FreeBSD, NetBSD, OpenBSD, and Solaris. On some systems it
even comes pre-installed.

Commands for installing zsh on the most popular platforms:

```sh
# Linux: Debian, Ubuntu, Mint, etc
sudo apt install zsh
# Linux: Fedora, RHEL, Rocky, etc
sudo dnf install zsh
# Linux: Arch, CachyOS, etc
sudo pacman -S zsh
# macOS: Homebrew
brew install zsh
```

For instructions on compiling zsh yourself, see `INSTALL`.
